# Contributing

## Local Development

```bash
npm install
npm test
npm run dev
```

## Pull Request Checklist

- Keep storage and retrieval behavior inspectable.
- Add a test when scoring, retrieval, or SDK behavior changes.
- Update README examples if the public API changes.
- Run `npm test` before opening a PR.

## Good First Issues

- Add an MCP server wrapper.
- Add a simple benchmark example with repeated agent runs.
- Add namespace/project support to isolate team data.
